# Chimera Runtime Skills

This directory contains runtime skill contracts for Project Chimera.

## Skills
- `skill_fetch_trends`: collects and normalizes trend data from source platforms.
- `skill_publish_openclaw_status`: publishes Chimera availability/status to OpenClaw.

Each skill README defines:
- Purpose and invocation context
- JSON input/output contracts
- Validation and exception behavior
- Side effects and non-goals
